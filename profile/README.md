# Multrapool...

...is the organization that makes modding for Ultrapool possible. you can be part of it too, if you contribute to any of our repositories :3 

PRs welcome!

-----

## How do I start playing with mods?

Add [Rakkit](https://github.com/Multrapool/Rakkit/wiki/Adding-Rakkit) :3 Then, put any mods you'd like inside your `mods` folder

## How do I start making mods?

Follow [this walkthrough](https://github.com/Multrapool/Cue/wiki/Setting-up-your-development-environment) to get your Godot project up and running :3
